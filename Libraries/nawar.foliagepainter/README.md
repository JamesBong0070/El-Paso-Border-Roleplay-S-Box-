# Foliage Painter

Paint foliage into your scene!

Some semi-wacky words are used to describe concepts based on the act of painting.

## Pigments

A pigment (ie type) is a single type of foliage that could be painter

## Palette

A collection of pigments (ie foliage types)

## Brush

How is the pigment from a palette painted onto the scene?

## Stroke

A stroke of a brush applies pigment (from a pallette) onto your scene

# How to use

1. Create an object in your scene, this will be your container
2. Select the object you created
3. Press `SHIFT+F` on your keyboard to enter Foliage painter mode
4. Choose a palette and start painting!
